### Laravel Project

Blog app

## Technologies and Algorithm

- Laravel Framework
- Tailwind
- CKEditor
- Database: MySQL

## Usage 

### Project Setup 
- Create file .env in the root folder
- Copy all in the .env.example to .env 
- Config your database in .env

### Install dependencies

```bash
  composer install
```

### Run project
- Run both command:

```bash
  php artisan serve
```

```bash
  npm run dev
```
